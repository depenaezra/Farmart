STEP-BY-STEP COPY COMMANDS (Linux/Mac):
================================================================================

# From ci4-application directory, run:

# 1. Copy Controllers
cp -r app/Controllers/* ../agriconnect-ci4/app/Controllers/

# 2. Copy Models
cp -r app/Models/* ../agriconnect-ci4/app/Models/

# 3. Copy Views
cp -r app/Views/* ../agriconnect-ci4/app/Views/

# 4. Copy Filters
cp -r app/Filters/* ../agriconnect-ci4/app/Filters/

# 5. Copy Routes
cp app/Config/Routes.php ../agriconnect-ci4/app/Config/Routes.php

# 6. Copy .htaccess
cp public/.htaccess ../agriconnect-ci4/public/.htaccess

# 7. Setup environment
cp env.example ../agriconnect-ci4/.env

# 8. Create upload folders
mkdir -p ../agriconnect-ci4/public/uploads/products
chmod -R 755 ../agriconnect-ci4/public/uploads

# 9. Import database
mysql -u root -p agriconnect < database/schema.sql
